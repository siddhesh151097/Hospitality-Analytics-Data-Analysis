# Hospitality Business Performance Analytics

## Project Overview
End-to-end data analysis project focused on hospitality industry performance metrics such as revenue, occupancy, cancellation rate, guest ratings, and booking trends using SQL, Power BI, Tableau, and Excel.

## Tools Used
- SQL
- Microsoft Excel
- Power BI
- Tableau

## Key KPIs
- Total Revenue
- Occupancy Percentage
- Cancellation Rate
- Revenue by City and Property
- Revenue by Booking Platform
- Guest Ratings

## Data Analysis Process
- Cleaned and validated raw data using Excel
- Performed KPI calculations using SQL
- Built data models for dashboards
- Designed interactive dashboards in Power BI and Tableau

## Business Insights
- Weekday bookings generate higher revenue compared to weekends
- Premium room categories drive a major portion of revenue
- Certain cities have high ratings but lower revenue potential
- Cancellation trends highlight operational improvement opportunities

## Folder Structure
- data: Raw and cleaned datasets
- sql: SQL queries
- powerbi: Power BI dashboard
- tableau: Tableau dashboard
- reports: Project documentation
